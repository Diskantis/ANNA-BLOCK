#nav {} -  селектор индетификатор: будут стилизованa все элементы к которым применен id="nav"

.nav {} -  селектор class: будут стилизованa все элементы к которым применен class="nav"

.grid-item a {}  - селектор контекста: будет стилизованa ссылка (a), в блоке (div) к которому пременен class="grid-item"
.grid-item h1 {}                       будет стилизован заголовок (h1), в блоке (div) к которому пременен class="grid-item"
.grid h2 span                          будет стилизованa строка (span), в заголовке (h2), в блоке (div) к которому пременен class="grid"

a {}- селектор элемента: будет стилизованы все ссылка
h1 {}                    будет стилизованы все заголовки (h1)

a + ul {} - селектор соседа: будет стилизованы ссылка (a) и список (ul)

ul > li {} - селектор потомка: будет стилизован только первый (li)

div::after {     - селектор использования псевдоэлементов:
    content: '';
}
div::before {
    content: '';
}
a:hover {} - селектор использования псевдоэлементов: будет стилизовать ссылку в другой цвет отличный от установленного ранее

.grit-item:nth-child(4) {} - селектор использования псевдоэлементов: будет стилизовать все, в 4 по порядку блоке (div) к которому пременен class="grit-item"
.grit-item:first-child {} - селектор использования псевдоэлементов: будет стилизовать все, в первом по порядку блоке (div) к которому пременен class="grit-item"
.grit-item:last-child {} - селектор использования псевдоэлементов: будет стилизовать все, в последнем по порядку блоке (div) к которому пременен class="grit-item"
